# Install Supoerpowers Workflow

Install superpowers workflow for Cursor and OpenCode.

Create a temporary shallow clone of https://github.com/obra/superpowers.git

## Install Location

- `ai/workflows/superpowers/skills`: all skills
- `ai/workflows/superpowers/agents`: agents definitions. No nested directory hierarchy.
- `ai/workflows/superpowers/commands`: commands definitions.

## Steps

- Add skills, agents, and commands from superpowers
- Review extracted skills, agents, and commands
